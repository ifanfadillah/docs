Below is an explanation of each widget/section in the **Dashboard Overview**:

---

### Total Inventory 
Shows the total number of inventories you currently own, including OOH (Out-of-Home) and DOOH (Digital-Out-of-Home).

<img
  src="/images/documentation/getting-started/introduction/dashboard-overview/dashboard-overview-total-inventory.png"
  alt="dashboard-overview-total-inventory"
  title="dashboard-overview-total-inventory"
  className="mx-auto"
  style={{ width:"50%" }}
/>

---

### Filter by Category Inventory
Filter to display the top 3 occupied inventories by type (DOOH or OOH).

---

### Filter by Range Date
Filter to display the top 3 occupied inventories by date range. By default, the last month is displayed.

---

### Reset Filter
Button to reset both category and date range filters.

---
