## How to Upload Creative Content

---

You can upload creative content in two ways, namely via your computer directory and URL link:

<Accordion
      title="via Computer"
>
<Steps>
      <Step title="Click the 'Upload' Button">
      
      ![Upload via Computer 1](/images/documentation/creative-management/media-library/upload-creative/upload-via-computer-1.png)

      Then it will display the Creative Content Upload.

      ![Upload via Computer 1.2](/images/documentation/creative-management/media-library/upload-creative/upload-via-computer-1.2.png)     
      </Step>
      <Step title="Click Drop File or Browse">
      ![Upload via Computer 2](/images/documentation/creative-management/media-library/upload-creative/upload-via-computer-2.png)     
      <Note>
            Ensure that the file size of the creative content to be uploaded does not exceed **50MB**.
      </Note>
            After selecting the file, it will display that the creative content file has been successfully added or uploaded.
      ![Upload via Computer 2.2](/images/documentation/creative-management/media-library/upload-creative/upload-via-computer-2.2.png)           
      </Step>
</Steps>
</Accordion>
<Accordion
      title="via URL Link"
>
<Steps>
      <Step title="Click the 'Upload' Button">
      
      ![Upload via URL Link 1](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-1.png)

      Then it will display the Creative Content Upload.

      ![Upload via URL Link 1.2](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-1.2.png)     
     
      </Step>
      <Step title="Click 'URL Link' Tab">
      
      ![Upload via URL Link 2](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-2.png)     
      
      Then the display will change to the upload tab via URL Link.
      
      ![Upload via Computer 2.2](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-2.2.png)           
      
      </Step>
      <Step title="Enter URL Link">
      Ensure that the URL Link entered is valid.

      ![Upload via URL Link 3](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-3.png)     
      
      </Step>
      <Step title="Enter Creative Duration">

      ![Upload via URL Link 4](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-4.png)     
      
      </Step>
      <Step title="Click the 'Save' Button">

      ![Upload via URL Link 5](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-5.png)     
      
      It will then display the URL of the creative content that has been successfully added or uploaded.
      
      ![Upload via URL Link 5.2](/images/documentation/creative-management/media-library/upload-creative/upload-via-url-link-5.2.png)     
      
      </Step>
</Steps>
</Accordion>
