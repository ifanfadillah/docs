For more flexibility in integration, Lynx SSP provides API Integration so that DOOH (Digital-Out-of-Home) Inventory can be connected to third-party systems.

This page will help you understand the DOOH (Digital-Out-of-Home) Inventory screen connection via API Integration.

## How to Connect using API Integration


---
<Info>
  Ensure you have added DOOH (Digital-Out-of-Home) Inventory on the Lynx SSP Platform ([How to Add New DOOH Inventory ↗](/documentation/inventory/dooh/add/how-to-add))
</Info>

<Steps>
  <Step title="Click the “Action” Button & Click “Edit”">
    ![How to Connect using API Integration 1](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-1.png)

    This will display the **Inventory Details Page**.

    ![How to Connect using API Integration 1.2](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-1.2.png)

  </Step>
  <Step title="Click “Screens” Tab">
    ![How to Connect using API Integration 2](/images/documentation/inventory/display/dooh/screens//api-integration/how-to-connect-using-api-2.png)

  </Step>
  <Step title="Click “Connect Player” Button">
    
    ![How to Connect using API Integration 3](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-3.png)

    Then it will display the connect player modal.

    ![How to Connect using API Integration 3.2](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-3.2.png)

  </Step>
  <Step title=" Click Connection Type drop-down menu and Select “API Integration”">
  
      ![How to Connect using API Integration 4](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-4.png)

  </Step>
    <Step title="Click “Save” Button">
  
    ![How to Connect using API Integration 5](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-5.png)

    Then API Integration is successfully connected and Copy API Secret Key for your production environment.

    ![How to Connect using API Integration 5.2](/images/documentation/inventory/display/dooh/screens/api-integration/how-to-connect-using-api-5.2.png) 
  </Step>
</Steps>
