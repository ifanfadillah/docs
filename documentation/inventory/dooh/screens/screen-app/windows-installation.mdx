Screen App is an application used to organize & display creative content that will be played on a Digital Billboard at a predetermined time and location through Lynx SSP. Screen App becomes a bridge between Lynx SSP (where you set up campaigns, billboards, etc.) and Media Player.

## How to Install ScreenApp on Windows?

## Minimum Requirement

| **OS**        | : | **Windows 10 (64-bit)**                      |
| :------------ | :-: | :------------------------------------------- |
| **Processor** | : | **Intel(R) Core(TM) i5-2520M CPU @ 2.50GHz** |
| **RAM**       | : | **8.00 GB (7.63 GB Usable)**                 |
| **SSD**       | : | **250 GB**                                   |
