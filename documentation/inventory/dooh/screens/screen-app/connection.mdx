When opened for the first time, the Screen App will display the Display ID which is six random characters inside a purple box. The Display ID is used for the binding process in the Lynx SSP Platform.

![Screen App 1](/images/documentation/inventory/display/dooh/screens/screen-app/connection/screen-app-1.png)

Click on the Gear logo in the top right corner.

![Screen App 2](/images/documentation/inventory/display/dooh/screens/screen-app/connection/screen-app-2.png)

You will be redirected to the Config UI as shown below. 

![Screen App 3](/images/documentation/inventory/display/dooh/screens/screen-app/connection/screen-app-3.png)

### Overview

 1. **Device ID** is a unique string of characters taken from the MAC address of the device.
 2. **Display ID** is a unique string of characters like on the main screen which will be used to bind to the inventory of the client portal.
 3. Custom height and custom width function to set the height and width of the frame from the player. The numbers listed do not represent sizes in any metric.
 4. **Screen** serves to select the screen (one device can run more than one screen).
 5. **Custom X and Custom Y** function to set the frame distance from the top left.
 6. Enable Resize if enabled, then you can adjust the size of the screen app, if turned off the screenapp size will be fixed and cannot be changed.
 7. Submit serves to apply configuration changes.
 8. Cancel serves to cancel configuration changes and return to the main Screen App screen.

There are several shortcuts that can be used on the Screen App as shown in the following table:

| **Shortcut** | **Description**                  |
| :----------- | :------------------------------- |
| **Alt + T**  | **Enable/Disable Always on Top** |
| **Esc**      | **Close full screen**            |
| **F**        | **Fullscreen**                   |
| **Alt + Q**  | **Close Apps**                   |

    Open the Screen App application and a Screen ID will appear that you can use to connect the billboard in Lynx SSP with the Screen App application.

      ![How to Connect using ScreenApp 2](/images/documentation/inventory/display/dooh/screens/screen-app/connection/screen-app-1.png)

      ![How to Connect using ScreenApp 3](/images/documentation/inventory/display/dooh/screens/screen-app/connection/how-to-connect-using-screen-app-3.png)

  </Step>
  <Step title="Click the drop-down menu for connection type and Select “ScreenApp”">
      ![How to Connect using ScreenApp 4](/images/documentation/inventory/display/dooh/screens/screen-app/connection/how-to-connect-using-screen-app-4.png)
  </Step>
  <Step title="Enter the screen ID obtained from the screen app.">
      ![How to Connect using ScreenApp 5](/images/documentation/inventory/display/dooh/screens/screen-app/connection/how-to-connect-using-screen-app-5.png)
  </Step>
  <Step title="Click the “Save” Button">
      ![How to Connect using ScreenApp 6](/images/documentation/inventory/display/dooh/screens/screen-app/connection/how-to-connect-using-screen-app-6.png)

    Then Screenapp is successfully connected.

      ![How to Connect using ScreenApp 6.2](/images/documentation/inventory/display/dooh/screens/screen-app/connection/how-to-connect-using-screen-app-6.2.png)

      You can perform the **connection check directly at the location of your DOOH (Digital-Out-of-Home) Inventory point**. Ensure that all previous steps have been completed and bind filler content to your DOOH (Digital-Out-of-Home) Inventory ([Create a Filler Content ↗](/documentation/creative-management/filler-content/create)). If you encounter any issues while performing the connection check, you can contact the Lynx Support team.
  </Step>
