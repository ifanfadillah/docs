The **Inventory Display** feature is a solution for users (Media Owners) to manage all of their billboard or display assets more effectively and in a structured manner. Through this feature, users can track, manage, and monitor their entire asset inventory in one place.

This page helps you understand the various menus available in the Inventory feature.

## Inventory Display Overview

![ Inventory Display Overview ](/images/documentation/inventory/display/overview/display-overview.png)


Based on the **Inventory Display Overview** above, here is an explanation of each section:

### Inventory Search
You can search for inventory by typing keywords based on the Inventory Name or SKU Number that you specified when creating it.


---

### Filter by Location Type
You can filter the inventory display data in the inventory list based on the location type. There are two location types you can choose from: **Indoor** and **Outdoor**.


---

### Filter by Type & Category
You can filter the inventory display data in the inventory list based on the type of inventory category you have, namely **DOOH (Digital-Out-of-Home)** & **OOH (Out-of-Home)** ([See Specification, Type & Category ↗](/documentation/inventory/specification-type-and-category)).


---

### Filter by Screen Status
You can filter the inventory display data in the inventory list based on the status of the screen that is connected to a display ([See Screen Status ↗]()).


---

### Add New Inventory Button
To create or register your inventory in the system, you can press the “Add New” button on this page, and the system will immediately take you to a new page where you must fill in the data regarding the specifications of the inventory ([See How to Add New Inventory]()).

![  Add New Inventory Button ](/images/documentation/inventory/display/overview/display-add-new-button.png)

---

### List Inventory Display
In this section, you can see all the inventory listings you have that have been successfully entered into the Lynx SSP portal. The properties that can be viewed in this table are as follows:

- Display Name
- SKU
- Rate 
- Screen Status ([See Screen Status ↗](/documentation/inventory/dooh/screens/status))
- Connection Type ([See Connection Type ↗](/documentation/inventory/dooh/screens/connect-to-screen#connection-type))
- Category ([See Specification, Type & Category ↗](/documentation/inventory/specification-type-and-category))

![  List Inventory Display ](/images/documentation/inventory/display/overview/display-list.png)

---

### Action Button "Edit" & "Delete" Inventory
Using the action button, you can navigate to the Edit Page ([See Edit Inventory ↗](/documentation/inventory/view-display/edit)) or delete Inventory ([See Delete Inventory ↗](/documentation/inventory/view-display/delete)) that has been added.

![ Action Button "Edit" & "Delete" Inventory ](/images/documentation/inventory/display/overview/display-action-button.png)

---

### Pagination
The sequential numbers in the image below are navigation buttons that you can use to go to the next inventory display page.

![ Pagination ](/images/documentation/inventory/display/overview/display-pagination.png)

---

### Page List
You can change the inventory list displayed. By default, it shows 10 items, but you can change it to 50 or 100.

![ Page List ](/images/documentation/inventory/display/overview/display-page-list.png)
