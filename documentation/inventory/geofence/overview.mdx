<Info>
   If you have Moving Ads Billboard media (mobile billboard media) such as Mobile LED, you can use this geofence feature to maximize the reach of your advertising content.   
</Info>

Geofence is a feature of Lynx SSP that can map a specific area or region using GPS or RFID technology. The region can be circular, square, or polygonal in shape to measure audience numbers.

## Geofence Overview

![ Geofence Overview ](/images/documentation/inventory/geofence/overview/geofence-overview.png)

Based on the **Geofence Overview** above, here is an explanation of each section:

---

### Search Geofence
You can search by geofence name has been registered.

---

### Create New Geofence Button
You can click the Create New button to add new geofence ([See Create a Geofence ↗](/documentation/inventory/geofence/create)).

![ Create New Geofence Button ](/images/documentation/inventory/geofence/overview/geofence-create-new-button.png)

---

### List Geofence
In this section, you can see all the geofence lists that you have and have successfully entered into the Lynx SSP portal.

![ List Geofence ](/images/documentation/inventory/geofence/overview/geofence-list.png)

---

### Hide Geofence Button
You can click the Hide button to hide your geofence from Map Area.

---

### Action Button "Edit" & "Delete"
Using the action button, you can navigate to the Edit Page or delete that has been created.

![ Action Button "Edit" & "Delete" ](/images/documentation/inventory/geofence/overview/geofence-action-button.png)

---

### Search Location
You also can search by location name.
