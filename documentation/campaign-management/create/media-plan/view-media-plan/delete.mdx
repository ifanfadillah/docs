---
title: "Delete a Media Plan"
---

<Info>
      You cannot delete media plan if status **"Launched"** and **"Finished"**
</Info>

You can delete your media plan by following these steps:

<Steps>
      <Step title="Click the “Delete” Button">

      ![Delete Media Plan 1](/images/documentation/campaign-management/create/media-plan/delete/delete-media-plan-1.png)

      </Step>
      <Step title="Click the “Yes, Delete” Button">

      ![Delete Media Plan 2](/images/documentation/campaign-management/create/media-plan/delete/delete-media-plan-2.png)
     
            You will then receive a notification that the media plan has been successfully deleted.
      </Step>
</Steps>
