---
title: "Edit a Media Plan"
---

If there is a change to the name of your media plan, you can change or edit it by following these steps:

<Steps>
      <Step title="Click the “Edit” Button">
            
      ![Edit Media Plan 1](/images/documentation/campaign-management/create/media-plan/edit/edit-media-plan-1.png)

            This will display the media plan name edit form.
      
      ![Edit Media Plan 1.2](/images/documentation/campaign-management/create/media-plan/edit/edit-media-plan-1.2.png)

      </Step>
      <Step title="Click the “Save Changes” Button">
 
            Once you have finished making changes, press the **“Save Changes”** button to save the modified data.
 
      ![Edit Media Plan 2](/images/documentation/campaign-management/create/media-plan/edit/edit-media-plan-2.png)
            
      </Step>
</Steps>
