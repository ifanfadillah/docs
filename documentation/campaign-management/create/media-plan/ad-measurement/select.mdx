<Info>
   Ad Measurement selection can only be done if the media plan has a **“Launch”** status or has already been launched ([See Launch Media Plan ↗](/documentation/campaign-management/create/media-plan/launch)).
</Info>

This page will help you understand Ad Measurement on the Lynx SSP Platform.

## How to Select Ad Measurement


---

Here are the steps for selecting Ad Measurement:

<Steps>
   <Step title="Click the “Inventories” tab in the Media Plan">
      
      ![How to Select Ad Measurement 1](/images/documentation/campaign-management/create/media-plan/ad-measurement/select/how-to-select-ad-measurement-1.png)

      Then it will display the inventories tab.

      ![How to Select Ad Measurement 1.2](/images/documentation/campaign-management/create/media-plan/ad-measurement/select/how-to-select-ad-measurement-1.2.png)

   </Step>
   <Step title="Click “Ad Measurement”">
      
      ![How to Select Ad Measurement 2](/images/documentation/campaign-management/create/media-plan/ad-measurement/select/how-to-select-ad-measurement-2.png)

   </Step>
   <Step title="Select Ad Measurement">
      You only can select one of Ad Measurement version ([See Ad Measurement Version ↗](./version))

      <Info>
         Some billboards are not compatible with the selected version method. Please review the compatibility list before proceeding.         
      </Info>

      ![How to Select Ad Measurement 3](/images/documentation/campaign-management/create/media-plan/ad-measurement/select/how-to-select-ad-measurement-3.png)

   </Step>
</Steps>
