---
title: "What is Shortlist"
---

The Shortlist feature functions as a dedicated workspace and selection list for the inventory you discover while exploring the platform. It acts as the crucial bridge between your initial map exploration and final campaign execution.

## Accessing Shortlist / Proposal Feature

You can access the shortlits feature by clicking the "**Shortlist**" Menu in the Navigation Bar.

![Accessing Shortlist 1](/images/accessing-shortlist-1.png)

Then it will display the **Shortlist Page.**

![Accessing Shortlist 1 2](/images/accessing-shortlist-1.2.png)

## Shortlist Overview

![Shortlist Overview 1](/images/shortlist-overview-1.png)

Based on the **Shortlist Overview** above, here is a detailed explanation of each section:

<Steps>
  <Step title="Search Bar">
    A text input field that allows you to quickly locate a specific shortlist. You can likely search by the shortlist's name or the name of the team member who created it.

    ![Shortlist Search Bar](/images/shortlist-search-bar.png)
  </Step>
  <Step title="Status Filter">
    A dropdown menu (currently showing "All Status") that lets you filter your view. You can use this to only display shortlists that are in a specific phase of your workflow, such as **"In Progress"** or **"Under Review"** (see Shortlist Status).

    ![Shortlist Status Filter](/images/shortlist-status-filter.png)
  </Step>
  <Step title="Create New Shortlist Button">
    The primary action button is used to generate a brand new, empty shortlist from scratch, allowing you to start a new media planning session (see Create New Shortlist).

    ![Shortlist Create New Button](/images/shortlist-create-new-button.png)
  </Step>
  <Step title="Shortlist Data Table">
    The main display area shows all your saved shortlists.

    ![Shortlist Create New Button 1](/images/shortlist-create-new-button-1.png)

    It provides a quick summary of each list, including:

    - **Name:** The designated title of the shortlist (e.g., Shortlist B).
    - **Created By:** The user or team member who generated the list.
    - **Inventory(s):** The total number of individual physical screens saved within that specific list.
    - **Total Price Estimation:** The combined estimated cost for all the inventories grouped in that shortlist.
    - **Last Update & Date Created:** Timestamps to help you track the latest modifications.
    - **Status:** A visual tag indicating the current progress of the list (e.g., "In Progress" or "Under Review").
  </Step>
  <Step title="Pagination">
    Navigation buttons ( `< 1 >` ) that allow you to flip through multiple pages of shortlists if your table exceeds the displayed limit.

    ![Shortlist Pagination](/images/shortlist-pagination.png)
  </Step>
  <Step title="Page List">
    A dropdown menu that lets you customize your view by selecting how many shortlist entries you want to see on a single page at one time (e.g., 10 per page).

    ![Shortlist Page List](/images/shortlist-page-list.png)
  </Step>
</Steps>
