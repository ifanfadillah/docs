---
title: "Remove Creative"
---

You can delete creative content that has been selected or added to each inventory in the previous media plan by following these steps:

<Steps>
      <Step title="Click the “Remove” Button">
            
            ![Remove Creative 1](/images/documentation/campaign-management/create/media-plan/creatives/remove/remove-creative-1.png)
      
      </Step>
      <Step title="Click the “Delete” Button">
            
            ![Remove Creative 2](/images/documentation/campaign-management/create/media-plan/creatives/remove/remove-creative-2.png)

            You will then receive a notification that the creative content has been successfully deleted.
      </Step>
</Steps>
